# 改变运行脚本的环境变量

在执行的命令中设置变量,区分测试和正式环境

主要依靠node的process.env

process 是node的全局对象

而process.env提供执行环境的环境变量

```shell
//改变命令的环境变量
$ export NODE_ENV=production && node app.js
# 或者
$ NODE_ENV=production node app.js
```

而在webpack中,就可以通过`process.env.NODE_ENV`获取变量
