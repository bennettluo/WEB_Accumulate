1. `npm init`: 创建新的package.json,加-f/-y(yes)可跳过
2. `npm set`: 设定环境变量,存在`~/.npmrc`文件

     2. init-author-email
     3. init-author-url
     4. init-license

##2. 其余命令

1. `npm help`: 获取npm帮助
2. `npm -l`: 获取npm的各种用法
3. `npm -v`: 查看npm版本
4. `npm config list -l`: npm配置
5. `npm install npm@latest -g`: 安装最新版的npm
