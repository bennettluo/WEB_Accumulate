# 快捷键

# 1. 分屏

1. 垂直分屏: command + d
2. 水平分屏: command + shift + d
3. 切换分屏: command + [ 或 ] / command + 左右方向键
4. 删除分屏: command + w 

# 2. 标签

1. 切换标签: command + 数字 / command + 左右方向
2. 切换全屏: command + Enter

# 3. 历史

1. 查看历史命令: command + ;
2. 查看剪贴板历史: command +shift + h
