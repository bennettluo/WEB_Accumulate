# position

CSS 之定位 POSTION

# static

默认值,没有定位,元素出现在正常的流中

top bottom lft right 或 z-index均被忽略

# 相对定位

`postion: relative`

存在正常流中 主要用于1absolute作为定位的依据

# 绝对定位

`position: absolute`

绝对定位, 绝对与谁? -- 绝对于父类不为static的元素
