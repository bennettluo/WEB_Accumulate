# Package.json

`package.json`主要用在npm的应用.

##1. 标签说明 
现在逐一解析它的标签

1. `script`: 指定执行命令的缩写.
     
     ```javascript
     "script": {
       "start": "node index.js"
       ...
     }
     ```
     
     运行`npm run start` 等价于 `node index.js`
2. `dependencies`: 项目运行依赖模块

    
3. `devDependencies`: 项目开发需要模块
